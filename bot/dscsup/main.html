Dsc Sup

Dsc Sup est un bot discord complétement innovant avec un concept jamais vus: Celui-ci a pour but de relier les serveurs volontaire en thermes d'annonce, il permet de relier les différentes annonces grace a une simple commande. Ce bot permet aussi aux membres de parler aux autres membres de différents sans être obliger de rejoindre les serveurs.

Liste des fonctionalitées :

    Comment être ajouter ?
  • /demande_admission: Envoie une demande au créateurs du bot
    • Utilisable par le créateur du serveur
  • /send [message]: Envoie votre annonce sur tout les serveurs partenaires
    • Exécutable par les fondateurs :
  • /addserveur [ID]: Définis le salon des annonces du serveur où le /send sera éxécuter.
  • /cochannel [ID]: Définis le salon de "cochannel" du serveur

    • Quel est la différence entre le bot et les salons annonces de discord ?

  • 1. Il y'a du délais entre la publication et la republications, le bot est obtimiser pour réduire le délais au max.
  • 2. Le bot publie un nombre ilimiter de serveur sans avoir de délais.
  • 3. Il est parfait pour informer les gens de vos évents.
  • 4. Le bot empèche automatiquement les pubs avec le /send.
